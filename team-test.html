📋 测试步骤指南

1. 启动开发服务器：运行 npm run dev 或 yarn dev
2. 访问个人中心页面：点击上方的测试链接或手动访问
3. 在个人中心页面，点击"🤝 团队中心"标签
4. 测试好友功能：在"👥 好友管理"标签中添加好友、处理请求
5. 测试聊天功能：点击好友卡片上的💬按钮开始聊天
6. 查看聊天记录：切换到"💬 聊天记录"标签查看历史对话
