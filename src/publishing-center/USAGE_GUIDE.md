# AllinONE 游戏发布系统 - 使用指南

## 系统概述

现在整个游戏发布系统已经完全可用！玩家可以：
1. 在游戏中心看到发布的游戏
2. 进入游戏游玩页面
3. 访问游戏专属商店
4. 购买道具并使用

---

## 完整使用流程

### 第一步：发布游戏（开发者）

```typescript
// 在 Publishing Center 页面
import { oneClickPublish } from '@/publishing-center';

// 一键发布
const result = await oneClickPublish({
  gameId: 'my-awesome-game',
  files: uploadedFiles,
  onProgress: (progress, message) => {
    console.log(`${progress}%: ${message}`);
  },
});

// 发布成功！
if (result.success) {
  console.log('游戏地址:', result.url);
  console.log('已启用Skills:', result.activatedSkills);
}
```

发布时会自动：
- ✅ AI分析游戏代码
- ✅ 推荐并注册Skills（Wallet、Inventory、Store）
- ✅ 生成游戏配置
- ✅ 保存游戏数据

---

### 第二步：游戏中心展示（玩家可见）

访问 `/game-center`，玩家会看到：

1. **内置游戏**（6个）
   - 消消乐、New Day、数字拼图、记忆翻牌、贪吃蛇、火柴人保卫战

2. **已发布的游戏**（显示在最下方）
   - 游戏名称
   - 框架图标
   - "已发布游戏"标签
   - **商店按钮** 🏪

---

### 第三步：进入游戏（玩家）

点击"游玩"按钮 → 进入 `/game/:gameId`

游戏页面显示：
- 🎮 游戏嵌入区域（iframe）
- 💰 实时余额显示
- 🔌 已启用的Skills状态
- 🏪 进入商店按钮
- 📊 游戏数据统计

---

### 第四步：游戏商店（玩家）

点击"商店"按钮 → 进入 `/game-store/:gameId`

商店功能：
- 🏷️ 商品分类：消耗品、武器、防具、高级
- 💎 余额显示：游戏币 + A币
- 🛒 购物车系统
- ⚡ 一键购买

**购买流程：**
```
1. 玩家点击"购买"
2. 系统检查余额
3. 扣除货币（Wallet Skill）
4. 添加道具到库存（Inventory Skill）
5. 显示"购买成功"
6. 余额实时更新
```

---

## 页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/game-center` | 游戏中心 | 展示所有游戏 |
| `/publishing-center` | 发布中心 | 发布新游戏 |
| `/game/:gameId` | 游戏游玩 | 嵌入游戏 + Skills |
| `/game-store/:gameId` | 游戏商店 | 购买道具 |

---

## Skills 集成说明

发布游戏时，系统会自动注册以下Skills到 SkillGateway：

| Skill | 功能 | 商店使用 |
|-------|------|----------|
| **Wallet** | 货币管理 | 检查余额、扣款 |
| **Inventory** | 道具库存 | 购买后添加道具 |
| **Store** | 商品管理 | 显示商品列表 |

---

## 示例：完整玩家旅程

```
1. 玩家访问 /game-center
   ↓
2. 看到新发布的 "勇者传说" 游戏
   ↓
3. 点击"游玩" → 进入 /game/勇者传说
   ↓
4. 看到游戏嵌入窗口 + 余额: 1000游戏币
   ↓
5. 点击"商店" → 进入 /game-store/勇者传说
   ↓
6. 看到商品：铁剑(100币)、生命药水(10币)
   ↓
7. 点击购买"铁剑"
   ↓
8. 余额扣减：1000 → 900
   ↓
9. 道具添加到库存
   ↓
10. 显示"购买成功！"
```

---

## 注意事项

1. **数据存储**：游戏数据存储在浏览器 localStorage，刷新页面不会丢失
2. **余额共享**：所有已发布游戏共享同一个钱包余额
3. **库存隔离**：每个游戏的道具是分开管理的
4. **匿名用户**：未登录玩家使用匿名账户，数据会保存
